# Thư mục chứa hình ảnh sản phẩm

## Yêu cầu:
- Cần tạo file `default-product.jpg` trong thư mục này
- File này sẽ được sử dụng làm hình ảnh mặc định cho sản phẩm không có hình

## Hướng dẫn:
1. Tìm hoặc tạo một hình ảnh placeholder (kích thước khuyến nghị: 400x400px)
2. Đặt tên file là `default-product.jpg`
3. Upload vào thư mục này

## Lưu ý:
- Các hình ảnh sản phẩm được upload sẽ tự động lưu trong thư mục này
- Định dạng: JPG, PNG, GIF
- Kích thước tối đa: 2MB
